# Drawing Warmups (Static)

A one-click classroom prompt generator for drawing warmups.

## Run locally
- Option 1: Use VS Code “Live Server”
- Option 2 (Python):
  - python3 -m http.server 8000
  - open http://localhost:8000

## Deploy on GitHub Pages
1. Push this repo to GitHub
2. Repo Settings → Pages
3. Source: Deploy from a branch
4. Branch: main / (root)
5. Save

Your site will be live at:
https://YOUR-USERNAME.github.io/YOUR-REPO/

## Data
Prompts are stored in `data/prompts.json`.
Add more prompts by appending objects to the JSON array.
